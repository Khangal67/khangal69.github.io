Unit 1: Data

This unit taught me how computers store and handle information. Some key terms:

  • bit: A binary digit (0 or 1).
  • byte: A sequence of 8 bits.
  • roundoff: Error from not enough bits to show a number fully.
  • analog data: Smoothly changing values (like sound waves).
  • lossless: Compression that keeps all data.
  • lossy: Compression that removes some data permanently.
  • metadata: Data about data (like a file‚Äôs details).

Unit 2: Computer Systems and Networks

This unit explained how computers and devices talk to each other across networks like the internet üåê.

  • Computing Device: A device that runs programs (laptops, phones).
  • Computer Network: A group of devices sharing information.
  • Path: The route data follows to reach its destination.
  • Protocol: Rules for communication between devices.
  • Fault Tolerant: Keeps working even if parts fail.
  • Scalability: Can grow without breaking.
  • Bandwidth: Maximum data sent per second.
  • URL: Human-friendly web address.
  • Packets: Small chunks of transmitted data.
  • Routers: Devices that direct packets.
  • DNS: Converts domain names to IP addresses.
