Unit 1: Data

  • bit: A binary digit, either 0 or 1.
  • byte: A sequence of 8 bits.
  • roundoff: Error when bits are not enough to represent numbers with full precision.
  • analog data: Values that change smoothly over time (e.g., pitch of music).
  • lossless: Compression that preserves all original data.
  • lossy: Compression that discards some data.
  • metadata: Data about data (e.g., file info).

Unit 2: Computer Systems and Networks

  • Computing Device: A physical device that can run a program.
  • Computer Network: Group of interconnected computing devices.
  • Path: Series of connections from sender to receiver.
  • Protocol: Rules for system communication.
  • Fault Tolerant: Keeps working even if parts fail.
  • Scalability: Ability to expand or contract to meet demands.
  • Bandwidth: Maximum transmission capacity.
  • URL: Easy-to-remember web address.
  • IP/TCP/UDP: Protocols for addressing and data transfer.
  • Packets: Small chunks of transmitted data.
  • Routers: Devices that forward data across networks.
  • DNS: Translates domain names into IP addresses.
  • HTTP/WWW: Protocol and system for web browsing.
